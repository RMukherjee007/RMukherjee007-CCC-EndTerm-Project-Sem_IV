# Real-World Algorithmic Solutions

A C++ demonstration of Greedy and Dynamic Programming paradigms applied to logistics and finance[cite: 2].

## Algorithms Covered
* **Greedy (Fractional Knapsack)**: Used for scenarios where items can be divided (e.g., loading raw materials)[cite: 2].
* **Dynamic Programming (0/1 Knapsack)**: Used for discrete "all-or-nothing" decisions (e.g., buying consumer electronics)[cite: 2].

## How to Run
1. Ensure you have a C++ compiler (like g++) installed[cite: 2].
2. Compile: `g++ main.cpp -o algo_demo`[cite: 2].
3. Run: `./algo_demo`[cite: 2].

## TUI Interaction
The program will launch an interactive menu allowing you to input custom capacities and budgets to see the algorithms work in real-time.
